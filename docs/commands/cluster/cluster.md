# Команды Cluster 🏠

Команды для управления локальным Kubernetes кластером (Kind) и секретами уровня кластера.
  
---

## 🏠 Локальный кластер (Local)

Управление локальным кластером на базе **Kind** осуществляется через группу команд `local`.

### Создание кластера

Поднимает локальный кластер, настраивает Ingress Nginx, применяет системные чарты и настраивает DNS.

  ```bash
  dodo cluster local create
  ```

#### Параметры
- `--config`, `-c`: Путь к кастомному конфигу kind.
- `--force`, `-f`: Сбросить кеш helm перед установкой системного чарта.
- `--version`, `-v`: Версия чарта `infra-paas`.

### Удаление кластера

Удаляет локальный кластер и очищает контексты.

  ```bash
  dodo cluster local delete
  ```

### Обновление токена

Обновляет токен для доступа к Yandex Container Registry внутри локального кластера.

  ```bash
  dodo cluster local refresh-token
  ```
**Важно:** Работает только в контексте локального кластера (`d-local*`).
